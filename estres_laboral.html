Descripción del Proyecto

Mood es una aplicación móvil diseñada para ayudar a los usuarios a gestionar el estrés laboral y prevenir el burnout. Ofrece herramientas como seguimiento del estado de ánimo, recordatorios de descanso, ejercicios de mindfulness y recomendaciones personalizadas.

Tecnologías Utilizadas

  • React Native (Desarrollo móvil)
  • Node.js (Backend)
  • MongoDB (Base de datos)
  • Figma (Diseño UX/UI)

Informe Completo: Mood - Gestión de Estrés Laboral

Resumen Ejecutivo: Mood es una aplicación móvil diseñada para ayudar a los usuarios a gestionar el estrés laboral y prevenir el burnout. Ofrece herramientas como seguimiento del estado de ánimo, recordatorios de descanso, ejercicios de mindfulness y recomendaciones personalizadas.

Problema a Resolver: El estrés laboral y el burnout son problemas crecientes que afectan la salud mental y la productividad de los trabajadores. Mood busca proporcionar una solución accesible y efectiva para mejorar el bienestar en el trabajo.

Objetivos del Proyecto:

  • Reducir los niveles de estrés y prevenir el burnout.
  • Mejorar la productividad y el bienestar de los usuarios.
  • Ofrecer herramientas personalizadas para gestionar el estrés.

Metodología:

  • Investigación de Usuario: Se realizaron encuestas y entrevistas para entender las necesidades de los usuarios.
  • Diseño UX/UI: Se diseñó una interfaz intuitiva y atractiva utilizando Figma.
  • Desarrollo: Se implementó la aplicación utilizando React Native para garantizar compatibilidad multiplataforma.
  • Pruebas y Validación: Se realizaron pruebas de usabilidad para garantizar una experiencia óptima.

Tecnologías Utilizadas:

  • React Native para el desarrollo móvil.
  • Node.js para el backend.
  • MongoDB para la base de datos.
  • Figma para el diseño UX/UI.

Resultados y Beneficios:

  • Reducción de los niveles de estrés entre los usuarios.
  • Mejora de la productividad y el bienestar en el trabajo.
  • Alta satisfacción del usuario con la interfaz y las funcionalidades.

Desafíos y Soluciones:

  • Personalización: Se implementaron algoritmos para ofrecer recomendaciones personalizadas basadas en el comportamiento del usuario.
  • Integración con Wearables: Se conectó la aplicación con dispositivos wearables para un seguimiento más preciso del estado de ánimo.
  • Privacidad de Datos: Se implementaron medidas de seguridad para proteger la información del usuario.

Conclusiones: Mood ha demostrado ser una herramienta efectiva para gestionar el estrés laboral y prevenir el burnout. La combinación de un diseño intuitivo y funcionalidades personalizadas ha permitido a los usuarios mejorar su bienestar y productividad en el trabajo.

Próximos Pasos:

  • Incorporar inteligencia artificial para recomendaciones más avanzadas.
  • Desarrollar una versión web de la aplicación.
  • Ampliar el alcance a más empresas y sectores.
