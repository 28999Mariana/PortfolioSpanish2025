Informe de Presentación: Reconocimiento Facial para Seguridad

Resumen Ejecutivo: Este proyecto implementó un sistema de reconocimiento facial utilizando redes neuronales convolucionales (CNN) para controlar el acceso a áreas restringidas en una empresa. El objetivo principal fue mejorar la seguridad y optimizar los tiempos de acceso.

Problema a Resolver: Las empresas enfrentan desafíos en la gestión de acceso a áreas sensibles, como la suplantación de identidad y los tiempos prolongados de verificación manual. Esto genera riesgos de seguridad y pérdida de eficiencia operativa.

Objetivos del Proyecto:

  • Mejorar la seguridad en áreas restringidas.
  • Reducir los tiempos de acceso mediante la automatización.
  • Registrar de manera precisa las entradas y salidas de los empleados.

Metodología:

  • Recopilación y Preprocesamiento de Datos: Se utilizan imágenes faciales de los empleados para entrenar el modelo.
  • Entrenamiento del Modelo: Se implementan redes neuronales convolucionales (CNN) utilizando TensorFlow y OpenCV.
  • Desarrollo de la Interfaz de Usuario: Se crea una interfaz en React para la gestión y visualización de los registros de acceso.
  • Pruebas y Validación: Se realizan pruebas de precisión y eficiencia en condiciones reales.

Tecnologías Utilizadas:

  • Python (OpenCV, TensorFlow).
  • Redes Neuronales Convolucionales (CNN).
  • React para la interfaz de usuario.
  • Figma para el diseño UX/UI.

Resultados y Impacto:

  • Mejora en la seguridad de las áreas restringidas.
  • Reducción en los tiempos de acceso.
  • Registro preciso de entradas y salidas en tiempo real.

Desafíos y Soluciones:

  • Precisión en la Identificación: Se optimizó el modelo CNN para reducir falsos positivos y negativos.
  • Integración con Sistemas Existentes: Se desarrollaron APIs para conectar el sistema con la infraestructura de la empresa.
  • Privacidad de Datos: Se implementaron medidas de cifrado y almacenamiento seguro de las imágenes.

Conclusiones: El sistema de reconocimiento facial desarrollado en este proyecto ha demostrado ser una solución efectiva para mejorar la seguridad y optimizar los procesos de acceso en entornos corporativos. La combinación de técnicas avanzadas de IA y una interfaz intuitiva ha permitido a la empresa cliente garantizar un entorno más seguro y eficiente.

Próximos Pasos:

  • Incorporar reconocimiento de emociones para mejorar la interacción con los empleados.
  • Expandir el sistema a otras áreas, como la gestión de visitantes.
  • Desarrollar una aplicación móvil para notificaciones en tiempo real.
