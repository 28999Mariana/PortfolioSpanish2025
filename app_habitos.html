Informe de Presentación: Aplicación Móvil para Gestión de Hábitos Saludables

Resumen Ejecutivo: Este proyecto consistió en el diseño y desarrollo de una aplicación móvil para ayudar a los usuarios a establecer y mantener hábitos saludables. La app incluye funcionalidades como recordatorios, seguimiento de progreso y recomendaciones personalizadas.

Problema a Resolver: Muchas personas luchan por mantener hábitos saludables debido a la falta de motivación, organización y seguimiento. Esta app busca resolver ese problema ofreciendo una herramienta integral y fácil de usar.

Objetivos del Proyecto:

  • Facilitar el establecimiento de metas relacionadas con hábitos saludables.
  • Proporcionar recordatorios y seguimiento de progreso.
  • Ofrecer recomendaciones personalizadas basadas en el comportamiento del usuario.

Metodología:

  • Investigación de Usuario: Se realizaron entrevistas y encuestas para entender las necesidades de los usuarios.
  • Diseño UX/UI: Se utilizaron Figma y Adobe XD para crear prototipos y diseñar la interfaz.
  • Pruebas de Usabilidad: Se llevaron a cabo pruebas con usuarios reales para validar el diseño.
  • Desarrollo: Se implementó la app utilizando React Native para garantizar compatibilidad multiplataforma.

Herramientas Utilizadas:

  • Figma (Diseño UX/UI).
  • Adobe XD (Prototipado).
  • User Testing (Pruebas de usabilidad).
  • React Native (Implementación del diseño).

Resultados y Impacto:

  • Más de 50,000 descargas en los primeros 3 meses.
  • Calificación de 4.8/5 en las tiendas de aplicaciones.
  • Mejora significativa en la adherencia a hábitos saludables entre los usuarios.

Desafíos y Soluciones:

  • Diseño Intuitivo: Se realizaron múltiples iteraciones de diseño basadas en feedback de usuarios.
  • Compatibilidad Multiplataforma: Se eligió React Native para garantizar un rendimiento óptimo en iOS y Android.
  • Personalización: Se implementaron algoritmos para ofrecer recomendaciones personalizadas.

Conclusiones: La aplicación móvil desarrollada en este proyecto ha demostrado ser una herramienta efectiva para ayudar a los usuarios a mantener hábitos saludables. La combinación de un diseño intuitivo y funcionalidades personalizadas ha resultado en una alta satisfacción del usuario y una adopción masiva.

Próximos Pasos:

  • Incorporar integración con wearables para un seguimiento más preciso.
  • Agregar funcionalidades de comunidad para fomentar la motivación grupal.
  • Expandir la app a nuevos mercados con soporte multilingüe.
